# 🧮 Simple Calculator

A beautifully designed and responsive web-based calculator built using HTML and CSS. This project is designed to perform basic arithmetic operations smoothly.

## 🚀 Features

- **Clean UI:** User-friendly interface with a modern look.
- **Basic Operations:** Supports Addition (+), Subtraction (-), Multiplication (*), and Division (/).
- **Reset Functionality:** A clear button (C) to reset the display instantly.
- **Responsive Design:** Works well across different screen sizes.

## 🛠️ Tech Stack

- **HTML5:** Used for creating the structure and layout of the calculator.
- **CSS3:** Used for styling, grid layout, colors, and smooth hover effects.
- *(Optional future update: **JavaScript** for handling the calculation logic).*
